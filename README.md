# mussolini
Sets your page in the typical dictator position!

![](oʇᴉuǝq.gif)

To install:

```
yarn add mussolini
```

To rotate the page:
```
import { mussoliniVisible } from 'mussolini'
mussoliniVisible()
```

To move Head below the Body:
```
import { mussoliniHidden } from 'mussolini'
mussoliniHidden()
```
